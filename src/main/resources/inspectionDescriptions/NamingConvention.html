Highlights identifiers that do not follow the required naming convention for references.

This inspection flags reference names that do not start with a lowercase letter or underscore, or that contain invalid characters such as uppercase letters or symbols. Following naming conventions helps keep code consistent and readable.

For example, the following names are not accepted: 


MyVariable   // starts with uppercase
foo-Bar      // contains a hyphen
1variable    // starts with a number

Accepted names follow this pattern: 


valid_name1
example2
_underscore
